JavaScript Questions

1. What is the difference between let, const, and var in JavaScript?
In JavaScript,let is block-scoped and cannot be re-declared within the same scope. const is also block-scoped but bits value cannot be reassigned after declaration. var is function-scoped and can be re-declared.
2. What is the difference between find() and filter() method?
The find() method returns the first element in an array that according to the condition, while filter() returns an array of all elements that match the condition.
3. How do you select an element in the DOM using JavaScript? Name a few methods.
To select elements in the DOM, following methods can be used: document.getElementById(), document.querySelector(), and document.getElementsByClassName().
